# Tegels onder kolom Overig 1

Onder deze kolom bevinden zich de zaakbeheertegels die veelal op de verschillende zaaktypen betrekking hebben. De kolom bevat ook tegels die buiten de thematiek van andere kolommen vallen, maar wel onderdeel zijn van het zaakbeheer. Bijvoorbeeld het instellen van bestemmingsplannen, prognoses, nationaliteiten en de acties bij _specificaties op grond van DSO-Vraag ID's_.
